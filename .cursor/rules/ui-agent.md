# UI Agent

Ты — агент UI-консистентности. Не предлагай новый дизайн — проверяй соответствие существующим паттернам.

## Чеклист кнопок

- [ ] Каждая кнопка имеет display:inline-flex; align-items:center; justify-content:center — через класс из index.css
- [ ] Кнопки используют классы: btn-primary, btn-ghost или btn-outline — не инлайн-стили для hover
- [ ] Нигде нет transform:scale() на hover — только box-shadow или background-color
- [ ] Нет кнопок без cursor:pointer

## Чеклист CSS-переменных

- [ ] Цвета фона: только var(--bg) или производные — не хардкоданные hex
- [ ] Цвета текста: var(--text) и var(--text-h) — не #333, не black
- [ ] Акцентный цвет: var(--accent) — не хардкоданный purple/violet
- [ ] Бордеры: var(--border) — не #eee, не #e5e5e5 напрямую
- [ ] Dark mode работает: проверь, что нет инлайн-цветов, которые не видны в тёмной теме

## Чеклист адаптивности

- [ ] Все блоки корректно отображаются на 375px (iPhone SE)
- [ ] Нет горизонтального скролла на мобильных
- [ ] Карточки в сетке: минимум 1 колонка на мобильном
- [ ] Размер кнопок достаточен для тапа: min-height 44px
- [ ] Модалка пароля не обрезается на мобильном

## Чеклист табов ("Предметные копилки")

- [ ] Активный таб визуально отличается от неактивных
- [ ] Таб-переключатель работает через клик, не hover
- [ ] На мобильном табы не выходят за пределы экрана — скролл или перенос

## Чеклист модалки пароля

- [ ] Есть backdrop (затемнённый фон за модалкой)
- [ ] Модалка закрывается по клику на backdrop
- [ ] Модалка закрывается по Escape
- [ ] Поле пароля имеет type="password"
- [ ] При неверном пароле есть сообщение об ошибке
- [ ] Кнопка submit задизейблена во время "проверки" (если есть debounce/async)

## Как проверять

1. Найди все <button> и <a> в компонентах — проверь классы
2. Найди все color: и background: — нет ли хардкоданных значений
3. Проверь медиа-запросы в index.css — покрыты ли все критические точки (768px, 375px)
4. Проверь модалку — backdrop, Escape, мобильный вид

## Если нашёл проблему

Укажи файл + класс/элемент. Предложи точечное исправление в index.css или компоненте.
Не трогай логику, не меняй структуру — только стили.

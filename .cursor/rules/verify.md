# Pre-commit верификация

Запускай агентов в порядке. Стоп при первом блокере.

## Шаг 1 — Логика и данные
@logic-agent
Проверь авторизацию, структуру данных, целостность состояния.
Блокер: любой баг в авторизации или утечка пароля из constants.js.

## Шаг 2 — UI-консистентность
@ui-agent
Проверь кнопки, CSS-переменные, адаптивность, модалку.
Блокер: сломанная мобильная вёрстка или хардкоданные цвета.

## Шаг 3 — Размер файлов
@refactor-agent
Найди файлы > 200 строк или смешанную ответственность.
Блокер: App.jsx > 150 строк с бизнес-логикой внутри.

## Шаг 4 — Сборка
Запусти: npm run lint && npm run build
Оба должны пройти без ошибок.

## Готово
Все шаги прошли → коммит.
Есть блокер → исправь, повтори с этого шага.

# UI Agent

Три цвета, два шрифта. Ищи любые отклонения.

## Палитра — единственно допустимая

Бирюзовый: #2ABFAA (основной), #1a9485 (тёмный), #e6faf7 (фон)
Розовый: #F0A8B8 (основной), #c97a8e (тёмный), #fdf0f3 (фон)
Белый: #FFFFFF — фон страницы и карточек
Текст: #1a2e2b (основной), #5a7470 (второстепенный)
Бордер: #d4ede9

Запрещено: любые другие hex, серые (#333, #666),
голубые не из палитры, чёрный (#000), system colors.

## Шрифты

Заголовки, номера секций, badges, pills → Cormorant Garamond
  (подключён через @font-face из src/assets/fonts/)
  Веса: 300 Light, 400 Regular, 500 Medium, 600 SemiBold, 700 Bold

Всё остальное → Nunito (Google Fonts)
  Веса: 400, 500, 600

Запрещено: system-ui, Arial, sans-serif как основные шрифты.
Запрещено: Cormorant Garamond для кнопок, табов, основного текста.
Запрещено: Nunito для заголовков h1, h2.

## Чеклист цветов

- [ ] Нет ни одного цвета вне палитры
- [ ] Фон страницы: #FFFFFF
- [ ] Карточки: белый фон + 1.5px solid #d4ede9
- [ ] Акцент: только #2ABFAA и производные
- [ ] Вторичное: только #F0A8B8 и производные
- [ ] Текст: только #1a2e2b или #5a7470

## Чеклист шрифтов

- [ ] h1, h2: Cormorant Garamond
- [ ] Номера секций ("01 —"): Cormorant Garamond Bold
- [ ] Badges, pills: Cormorant Garamond
- [ ] Кнопки, табы, подписи, основной текст: Nunito
- [ ] @font-face подключены в index.css
- [ ] Nunito подключён в index.html через Google Fonts

## Чеклист компонентов

- [ ] Primary button: #2ABFAA фон, белый текст, radius 8px, Nunito
- [ ] Teal outline button: #e6faf7 + 1.5px #2ABFAA + #1a9485 текст, Nunito
- [ ] Pink outline button: #fdf0f3 + 1.5px #F0A8B8 + #c97a8e текст, Nunito
- [ ] Активный таб: underline 2px solid #2ABFAA
- [ ] Пунктир блока копилок: 2px dashed #F0A8B8
- [ ] SVG-стрелка на лендинге: stroke #2ABFAA

## Чеклист лендинга

- [ ] Правая панель: полосы #B8DFF8 / #EAF5FF по 18px
- [ ] Разделитель секций: 2px dashed #F0A8B8
- [ ] Нет градиентов, теней, blur

## Чеклист адаптивности

- [ ] 375px: нет горизонтального скролла
- [ ] Карточки: 1 колонка на мобильном, 2 на десктопе
- [ ] Табы не выходят за экран на 375px
- [ ] Кнопки: min-height 44px

## Формат ответа при нарушении

Файл + строка: что нарушено → как исправить.
Не трогай логику, только стили.
